### Support for Fluent Bit 0.8.5

Users will now receive Fluent Bit version 0.8.5, which includes fixes for known vulnerabilities. For more details, refer  https://fluentbit.io/announcements/.

### Changed

* Infrastructure agent recommends Fluent Bit 0.8.5 packages.
* [Fluent Bit Output Plugin Docker image](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) uses Fluent Bit 0.8.5.
* [New Relic Logging Helm chart](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) installs the new plugin image.
